Error: That port is already in use.

Причина: Какой то процесс уже слушает порт и не дает к нему подключиться.

Лечение: Убиваем процесс по порту.

netstat -pl |grep 8000

Видим нечто такого.

tcp        0      0 localhost:8000          *:*                     LISTEN      14977/python

Прибиваем по ID процесса

Установка и сборка nginx.

Date ср 11 лютий 2015 By zdimon77@gmail.con Category Разное. Tags python tutorial tests

Забираем ключик.

mkdir nginx_source
cd nginx_source
wget http://nginx.org/keys/nginx_signing.key

Импортируем его в сисему.

sudo apt-key add nginx_signing.key

Добавим репу.

sudo nano /etc/apt/sources.list

В конец файла.

deb http://nginx.org/packages/ubuntu/ trusty nginx
deb-src http://nginx.org/packages/ubuntu/ trusty nginx
