Работаем с репозиторием git

Date нд 08 лютий 2015 By zdimon77@gmail.con Category Урок 1. Tags python уроки курсы

Регистрируемся на сайте https://github.com Создаем репозиторий.

Установка git.

apt-get install git

Создадим каталог приложения.

mkdir myproject

Инициализация репозитория.

cd myproject
git init

Создадим файл с кодом.

echo 'print "Hello world"' >> hello.py

Добавим файл для комита.

git add .

Просмотрим статус.

git status

Закомитим изменения локально.

git commit -m ...
