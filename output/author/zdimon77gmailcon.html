Часто возникающие ошибки

Date ср 11 лютий 2015 By zdimon77@gmail.con Category Ошибки. Tags python tutorial уроки django

Error: That port is already in use.

Причина: Какой то процесс уже слушает порт и не дает к нему подключиться.

Лечение: Убиваем процесс по порту.

netstat -pl |grep 8000

Видим нечто такого.

tcp        0      0 localhost:8000          *:*                     LISTEN      14977/python

Прибиваем по ID процесса

kill -9 14977
